# Experiment 10: Employee Salary Update Validation Trigger

## Aim
To create a `BEFORE UPDATE` trigger on the `Employee` table in MySQL that validates updates to the salary column, raising a custom error if a user attempts to update a salary to any value equal to or greater than `1000`.

---

## 1. Database & Table Setup

Copy and run this block in your MySQL command line or GUI to set up the database and the table.

```sql
-- Step 1: Create and use a new database
CREATE DATABASE IF NOT EXISTS dbms_lab_q10;
USE dbms_lab_q10;

-- Step 2: Drop existing tables
DROP TABLE IF EXISTS Employee;

-- Step 3: Create the Employee table
CREATE TABLE Employee (
    emp_id INT PRIMARY KEY AUTO_INCREMENT,
    e_name VARCHAR(100) NOT NULL,
    salary DECIMAL(10, 2) NOT NULL
);

-- Step 4: Insert sample employees with salaries less than 1000
INSERT INTO Employee (e_name, salary) VALUES
('Ravi Kumar', 750.00),
('Seema Das', 900.00),
('Joy Sen', 500.00);
```

### Initial Employee Table
After inserting, the table looks like this:

| emp_id | e_name | salary |
|---|---|---|
| 1 | Ravi Kumar | 750.00 |
| 2 | Seema Das | 900.00 |
| 3 | Joy Sen | 500.00 |

---

## 2. Trigger Creation

This `BEFORE UPDATE` trigger runs automatically right before any update statement takes place on the `Employee` table. It will inspect the incoming new salary and block it if it exceeds the limit.

#### SQL Statement to Create Trigger:
```sql
DELIMITER //

CREATE TRIGGER before_employee_salary_update
BEFORE UPDATE ON Employee
FOR EACH ROW
BEGIN
    -- Check if the new salary being set is 1000 or greater
    IF NEW.salary >= 1000.00 THEN
        SIGNAL SQLSTATE '45000'
        SET MESSAGE_TEXT = 'ERROR: Update blocked. Employee salary must be strictly less than 1000.';
    END IF;
END //

DELIMITER ;
```

---

## 3. Testing and Verification

### Scenario 1: Update an employee's salary to a valid amount (< 1000)

#### SQL Query:
```sql
UPDATE Employee 
SET salary = 950.00 
WHERE e_name = 'Ravi Kumar';
```

#### Verification:
Run this select query to view the table:
```sql
SELECT * FROM Employee;
```

#### Output:
| emp_id | e_name | salary |
|---|---|---|
| 1 | Ravi Kumar | 950.00 |
| 2 | Seema Das | 900.00 |
| 3 | Joy Sen | 500.00 |

*(Note: The update succeeded because 950.00 is less than 1000).*

---

### Scenario 2: Try to update an employee's salary to an invalid amount (>= 1000)

#### SQL Query:
```sql
UPDATE Employee 
SET salary = 1200.00 
WHERE e_name = 'Seema Das';
```

#### Output/Result:
`ERROR 1644 (45000): ERROR: Update blocked. Employee salary must be strictly less than 1000.`

*(Note: MySQL successfully blocked the update and Seema's salary remains unchanged at 900.00).*
