## Overview

This folder contains the Database Management System (DBMS) Lab Practical Queries and Databases in MySQL, implemented as part of the curriculum for the **Bachelor of Technology (B.Tech) in Computer Science and Engineering (CSE), 4th Semester** at **Royal Global University**.

Each folder experiment features a complete, self-explanatory **Markdown (.md) guide** designed specifically for beginners. These guides include:
* Full database schema definitions.
* Complete copy-paste SQL codeblocks to initialize tables and load data locally at once.
* Simple, basic query-by-query solutions that are easy to understand.
* Expected outputs represented as markdown tables.

---

## How to Run locally (Query-by-Query)

To run these queries on your local machine:
1. Open your MySQL Command Line Client or your GUI of choice (like MySQL Workbench, phpMyAdmin, DBeaver, etc.).
2. Go to the markdown file of the desired experiment listed in the table below.
3. Copy the **Database & Table Setup** SQL block first and paste it into MySQL to load the database and insert the sample records.
4. Copy and execute each individual query one by one to see the results.
