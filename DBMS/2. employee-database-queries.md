# Experiment 2: Employee Database & Basic Queries

## Aim
To create an `Employee` table and perform basic data retrieval queries based on department, salary ranges, and job designation in MySQL.

---

## 1. Database & Table Setup

Copy and run this block in your MySQL command line or GUI to set up the database, create the table, and insert the sample data.

```sql
-- Step 1: Create and use a new database
CREATE DATABASE IF NOT EXISTS dbms_lab_q2;
USE dbms_lab_q2;

-- Step 2: Drop table if it already exists (to start fresh)
DROP TABLE IF EXISTS Employee;

-- Step 3: Create the Employee table
CREATE TABLE Employee (
    emp_id INT PRIMARY KEY,
    e_name VARCHAR(100) NOT NULL,
    designation VARCHAR(50) NOT NULL,
    salary DECIMAL(10, 2) NOT NULL,
    dept VARCHAR(50) NOT NULL
);

-- Step 4: Insert sample records into the Employee table
```

## 2. Queries and Solutions

You can run each query one by one on your local machine.

### i) Display the details of employees working in the HR department.

#### SQL Query:
```sql
SELECT * 
FROM Employee 
WHERE dept = 'HR';
```

#### Output:
| emp_id | e_name | designation | salary | dept |
|---|---|---|---|---|
| 101 | Amit Sharma | Manager | 75000.00 | HR |
| 103 | Rahul Verma | Executive | 25000.00 | HR |
| 107 | Arjun Kumar | Manager | 65000.00 | HR |

#### Explanation:
* `SELECT *` means we want to see all columns.
* `WHERE dept = 'HR'` filters the rows to show only those where the department is exactly `'HR'`.

---

### ii) Display the names of employees who are getting a salary within the range 20k to 50k.

#### SQL Query:
```sql
SELECT e_name, salary 
FROM Employee 
WHERE salary BETWEEN 20000.00 AND 50000.00;
```

#### Output:
| e_name | salary |
|---|---|
| Priya Patel | 45000.00 |
| Rahul Verma | 25000.00 |
| Vikram Singh | 35000.00 |

#### Explanation:
* We select only the columns `e_name` and `salary`.
* The `BETWEEN` operator checks if the salary is inclusive of the values `20000` and `50000`.

---

### iii) Display the IDs of employees who is a manager and getting salary more than 60k.

#### SQL Query:
```sql
SELECT emp_id, e_name, salary 
FROM Employee 
WHERE designation = 'Manager' AND salary > 60000.00;
```

#### Output:
| emp_id | e_name | salary |
|---|---|---|
| 101 | Amit Sharma | 75000.00 |
| 107 | Arjun Kumar | 65000.00 |

#### Explanation:
* `AND` is used to combine two conditions. Both conditions must be true:
  1. The `designation` must be `'Manager'`.
  2. The `salary` must be strictly greater than `60000.00` (so `60k` is not included, only above it).
