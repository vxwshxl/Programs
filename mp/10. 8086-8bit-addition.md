# 8086 8 Bit Addition

## Aim
To add two 8-bit numbers using the 8086 microprocessor.

## Algorithm
1. Load first number into AL register.
2. Load second number into BL register.
3. Add BL to AL.
4. Store result at memory location 2502H.
5. Stop execution.

## Program
```asm
MOV AL,[2500]
MOV BL,[2501]
ADD AL,BL
MOV [2502],AL
HLT
```

## Machine Code

| Instruction | Code |
|-------------|------|
| MOV AL,[2500] | A0 00 25 |
| MOV BL,[2501] | 8A 1E 01 25 |
| ADD AL,BL | 00 D8 |
| MOV [2502],AL | A2 02 25 |
| HLT | F4 |

## Example Memory Values

| Address | Value |
|----------|-------|
| 2500 | 05 |
| 2501 | 03 |

## Output

After execution:

| Address | Result |
|----------|--------|
| 2502 | 08 |

## Explanation
- `MOV` transfers data.
- `ADD` performs addition.
- Result is stored in memory.
- `HLT` stops execution.
