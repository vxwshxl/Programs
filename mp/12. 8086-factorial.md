# 8086 Factorial of a Number

## Aim
To find the factorial of a number using the 8086 microprocessor.

## Algorithm
1. Load the number into CL register.
2. Initialize AX register with 0001H.
3. Multiply AX with CL.
4. Decrement CL.
5. Repeat multiplication until CL becomes zero.
6. Stop execution.

## Program
```asm
MOV CL,05H
MOV AX,0001H

BACK: MUL CL
DEC CL
JNZ BACK

HLT
```

## Example Input

| Register | Value |
|-----------|-------|
| CL | 05H |

## Output

Factorial of 5:

| Register | Result |
|-----------|--------|
| AX | 0078H |

## Explanation
- `MOV` loads initial values.
- `MUL CL` multiplies AX by CL.
- `DEC CL` decreases the counter.
- `JNZ` repeats loop until CL becomes zero.
- `HLT` stops execution.
