# 8085 16 Bit Subtraction

## Aim
To subtract two 16-bit numbers using the 8085 microprocessor.

## Algorithm
1. Load the first 16-bit number from memory location 2500H–2501H.
2. Exchange HL and DE register pairs.
3. Load the second 16-bit number from memory location 2502H–2503H.
4. Subtract lower bytes.
5. Subtract higher bytes with borrow.
6. Store the result at memory location 2504H–2505H.
7. Stop execution.

## Program
```asm
LHLD 2500H
XCHG
LHLD 2502H
MOV A,L
SUB E
MOV L,A
MOV A,H
SBB D
MOV H,A
SHLD 2504H
HLT
```

| Address | Result |
|----------|--------|

## Explanation
- `LHLD` loads 16-bit data into HL.
- `XCHG` exchanges HL and DE.
- `SUB` subtracts lower byte.
- `SBB` subtracts higher byte including borrow.
- `SHLD` stores the result.
- `HLT` stops execution.
