# 8086 8 Bit Subtraction

## Aim
To subtract two 8-bit numbers using the 8086 microprocessor.

## Algorithm
1. Load first number into AL register.
2. Load second number into BL register.
3. Subtract BL from AL.
4. Store result at memory location 2502H.
5. Stop execution.

## Program
```asm
MOV AL,[2500]
MOV BL,[2501]
SUB AL,BL
MOV [2502],AL
HLT
```

## Example Memory Values

| Address | Value |
|----------|-------|
| 2500 | 08 |
| 2501 | 03 |

## Output

After execution:

| Address | Result |
|----------|--------|
| 2502 | 05 |

## Explanation
- `MOV` transfers data into registers.
- `SUB` subtracts BL from AL.
- Result is stored in memory.
- `HLT` stops execution.
