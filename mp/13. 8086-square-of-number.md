# 8086 Square of a Number

## Aim
To find the square of a number using the 8086 microprocessor.

## Algorithm
1. Load number into AL register.
2. Multiply AL by itself.
3. Store result in AX register.
4. Stop execution.

## Program
```asm
MOV AL,05H
MUL AL
HLT
```

## Example Input

| Register | Value |
|-----------|-------|
| AL | 05H |

## Output

After execution:

| Register | Result |
|-----------|--------|
| AX | 0019H |

## Explanation
- `MOV` loads the number into AL.
- `MUL AL` multiplies AL by itself.
- Result is stored in AX register.
- `HLT` stops execution.
