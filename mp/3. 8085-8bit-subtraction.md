# 8085 8 Bit Subtraction

## Aim
To subtract two 8-bit numbers using the 8085 microprocessor.

## Algorithm
1. Load the first number from memory location 2500H.
2. Store it in register B.
3. Load the second number from memory location 2501H.
4. Subtract the first number from the second number.
5. Store the result at memory location 2502H.
6. Stop execution.

## Program
```asm
LDA 2500H
MOV B,A
LDA 2501H
SUB B
STA 2502H
HLT
```

## Machine Code

| Address | Code |
|----------|------|
| 2000 | 3A |
| 2001 | 00 |
| 2002 | 25 |
| 2003 | 47 |
| 2004 | 3A |
| 2005 | 01 |
| 2006 | 25 |
| 2007 | 90 |
| 2008 | 32 |
| 2009 | 02 |
| 200A | 25 |
| 200B | 76 |

## Example Memory Values

| Address | Value |
|----------|-------|
| 2500 | 03 |
| 2501 | 08 |

## Output

After execution:

| Address | Result |
|----------|--------|
| 2502 | 05 |

## Explanation
- `LDA` loads data into the accumulator.
- `MOV B,A` stores the first number in register B.
- `SUB B` subtracts register B from accumulator.
- `STA` stores the result in memory.
- `HLT` stops execution.
