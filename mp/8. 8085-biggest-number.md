# 8085 Biggest Number

## Aim
To find the biggest number among a set of numbers using the 8085 microprocessor.

## Algorithm
1. Load starting memory address into HL pair.
2. Load first number into accumulator.
3. Set counter.
4. Compare accumulator with next number.
5. Replace accumulator if larger number is found.
6. Repeat until all numbers are checked.
7. Store largest number at memory location 2600H.
8. Stop execution.

## Program
```asm
LXI H,2500H
MOV A,M
MVI C,04H

LOOP: INX H
CMP M
JNC SKIP
MOV A,M

SKIP: DCR C
JNZ LOOP

STA 2600H
HLT
```

## Example Memory Values

| Address | Value |
|----------|-------|
| 2500 | 25 |
| 2501 | 45 |
| 2502 | 12 |
| 2503 | 67 |
| 2504 | 40 |

## Output

After execution:

| Address | Result |
|----------|--------|
| 2600 | 67 |

## Explanation
- `CMP M` compares accumulator with memory value.
- `JNC` skips update if accumulator is larger.
- `MOV A,M` updates accumulator with larger value.
- `STA` stores biggest number.
