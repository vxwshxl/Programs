# 8085 8 Bit Addition

## Aim
To add two 8-bit numbers using the 8085 microprocessor.

## Algorithm
1. Load the first number from memory location 2500H.
2. Store it in register B.
3. Load the second number from memory location 2501H.
4. Add the first number to the second number.
5. Store the result at memory location 2502H.
6. Stop execution.

## Program
```asm
LDA 2500H
MOV B,A
LDA 2501H
ADD B
STA 2502H
HLT
```

## Machine Code

| Address | Code |
|----------|------|
| 2000 | 3A |
| 2001 | 00 |
| 2002 | 25 |
| 2003 | 47 |
| 2004 | 3A |
| 2005 | 01 |
| 2006 | 25 |
| 2007 | 80 |
| 2008 | 32 |
| 2009 | 02 |
| 200A | 25 |
| 200B | 76 |

## Example Memory Values

| Address | Value |
|----------|-------|
| 2500 | 05 |
| 2501 | 03 |

## Output

After execution:

| Address | Result |
|----------|--------|
| 2502 | 08 |

## Explanation
- `LDA` loads data into the accumulator.
- `MOV B,A` stores the first number in register B.
- `ADD B` adds register B to the accumulator.
- `STA` stores the final result in memory.
- `HLT` stops program execution.
