# 8086 Smallest of 5 Numbers

## Aim
To find the smallest number among five numbers using the 8086 microprocessor.

## Algorithm
1. Load starting memory address into SI register.
2. Load first number into AL register.
3. Set loop counter.
4. Compare AL with next number.
5. Update AL if smaller number is found.
6. Repeat until all numbers are checked.
7. Store smallest number at memory location 2600H.
8. Stop execution.

## Program
```asm
MOV SI,2500H
MOV AL,[SI]
MOV CX,0004H

BACK: INC SI
CMP AL,[SI]
JC SKIP
MOV AL,[SI]

SKIP: LOOP BACK

MOV [2600H],AL
HLT
```

## Example Memory Values

| Address | Value |
|----------|-------|
| 2500 | 25 |
| 2501 | 45 |
| 2502 | 12 |
| 2503 | 67 |
| 2504 | 40 |

## Output

After execution:

| Address | Result |
|----------|--------|
| 2600 | 12 |

## Explanation
- `CMP` compares values.
- `JC` skips update if AL is already smaller.
- `MOV AL,[SI]` updates AL with smaller value.
- `LOOP` continues until counter becomes zero.
- Result is stored at 2600H.
