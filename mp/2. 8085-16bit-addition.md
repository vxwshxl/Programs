# 8085 16 Bit Addition

## Aim
To add two 16-bit numbers using the 8085 microprocessor.

## Algorithm
1. Load the first 16-bit number from memory location 2500H–2501H.
2. Exchange the contents of HL and DE registers.
3. Load the second 16-bit number from memory location 2502H–2503H.
4. Add the contents of DE register pair to HL register pair.
5. Store the result at memory location 2504H–2505H.
6. Stop execution.

## Program
```asm
LHLD 2500H
XCHG
LHLD 2502H
DAD D
SHLD 2504H
HLT
```

## Machine Code

| Address | Code |
|----------|------|
| 2000 | 2A |
| 2001 | 00 |
| 2002 | 25 |
| 2003 | EB |
| 2004 | 2A |
| 2005 | 02 |
| 2006 | 25 |
| 2007 | 19 |
| 2008 | 22 |
| 2009 | 04 |
| 200A | 25 |
| 200B | 76 |

## Example Memory Values

### First 16-bit Number

| Address | Value |
|----------|-------|
| 2500 | 34 |
| 2501 | 12 |

### Second 16-bit Number

| Address | Value |
|----------|-------|
| 2502 | 78 |
| 2503 | 56 |

## Output

After execution:

| Address | Result |
|----------|--------|
| 2504 | AC |
| 2505 | 68 |

## Explanation
- `LHLD` loads a 16-bit value into HL register pair.
- `XCHG` exchanges HL and DE register pairs.
- `DAD D` adds DE register pair to HL register pair.
- `SHLD` stores the 16-bit result in memory.
- `HLT` stops program execution.
