# 8085 8 Bit Division

## Aim
To divide two 8-bit numbers using repeated subtraction in the 8085 microprocessor.

## Algorithm
1. Load dividend from memory location 2500H.
2. Store it in register B.
3. Load divisor from memory location 2501H.
4. Store it in register C.
5. Initialize register D as 00H to store quotient.
6. Repeatedly subtract divisor from dividend.
7. Increment quotient after each subtraction.
8. Stop when carry occurs.
9. Store quotient at memory location 2502H.
10. Stop execution.

## Program
```asm
LDA 2500H
MOV B,A
LDA 2501H
MOV C,A
MVI D,00H

LOOP: SUB C
JC END
INR D
JMP LOOP

END: MOV A,D
STA 2502H
HLT
```

## Example Memory Values

| Address | Value |
|----------|-------|
| 2500 | 0C |
| 2501 | 03 |

## Output

After execution:

| Address | Result |
|----------|--------|
| 2502 | 04 |

## Explanation
- `SUB C` subtracts divisor from accumulator.
- `JC END` checks for borrow and ends loop.
- `INR D` increments quotient.
- `MOV A,D` moves quotient to accumulator.
- `STA` stores quotient in memory.
- `HLT` stops execution.
