# 8085 Addition of 10 Numbers

## Aim
To add ten numbers using the 8085 microprocessor.

## Algorithm
1. Load starting memory address into HL pair.
2. Set counter as 10.
3. Clear accumulator.
4. Add each memory value to accumulator.
5. Increment memory pointer.
6. Decrement counter.
7. Repeat until counter becomes zero.
8. Store result at memory location 2600H.
9. Stop execution.

## Program
```asm
LXI H,2500H
MVI C,0AH
MVI A,00H

LOOP: ADD M
INX H
DCR C
JNZ LOOP

STA 2600H
HLT
```

## Example Memory Values

| Address | Value |
|----------|-------|
| 2500 | 01 |
| 2501 | 02 |
| 2502 | 03 |
| 2503 | 04 |
| 2504 | 05 |
| 2505 | 06 |
| 2506 | 07 |
| 2507 | 08 |
| 2508 | 09 |
| 2509 | 0A |

## Output

After execution:

| Address | Result |
|----------|--------|
| 2600 | 37 |

## Explanation
- `LXI H` loads memory pointer.
- `ADD M` adds memory content to accumulator.
- `INX H` moves to next memory location.
- `DCR C` decreases counter.
- `JNZ` repeats loop until counter reaches zero.
- `STA` stores final sum.
