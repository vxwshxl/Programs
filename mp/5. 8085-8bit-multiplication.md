# 8085 8 Bit Multiplication

## Aim
To multiply two 8-bit numbers using repeated addition in the 8085 microprocessor.

## Algorithm
1. Load the first number from memory location 2500H.
2. Store it in register B.
3. Load the second number from memory location 2501H.
4. Store it in register C.
5. Clear accumulator.
6. Add the first number repeatedly until counter becomes zero.
7. Store the result at memory location 2502H.
8. Stop execution.

## Program
```asm
LDA 2500H
MOV B,A
LDA 2501H
MOV C,A
MVI A,00H

LOOP: ADD B
DCR C
JNZ LOOP

STA 2502H
HLT
```

## Explanation
- `MOV C,A` stores multiplier in register C.
- `MVI A,00H` clears accumulator.
- `ADD B` performs repeated addition.
- `DCR C` decreases counter.
- `JNZ LOOP` repeats until zero.
- `STA` stores final result.
- `HLT` stops execution.
